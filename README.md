# laravel-todo
laravelで作成したTODOアプリ(TechSelect)

# 起動手順
1.Docker Desktopの起動

2.vscodeでプロジェクトを開く

3.ターミナルで以下のコマンドを実行

```
docker compose up
```

4.localhost:8080にアクセス

5.終了するには`ctrlキー`+`cキー`
